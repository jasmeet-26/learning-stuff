Mastering Flexbox

There are 2 types of access : A)Main access B)Other access. For the web the main access in flexbox goes from LEFT to RIGHT --> Horizontally

Flexbox is a way in CSS to arrange items inside a box. When you turn it on using display: flex, all the items inside the box line up in a row by default. It helps you control how those items move, grow, shrink, or align — without needing complicated code. It's useful when you're trying to place things next to each other or space them out evenly.

1.Default Flex(Row)

The default flex direction is row from left to right. By default: Items are placed horizontally (in a row). They do not wrap to the next line (they may shrink instead).

3.Justify Content- space between

justify-content controls how flex items are spaced along the main axis.
Common values: flex-start --> items start at the beginning
flex-end --> items at the end
center --> items in the center
space-between --> equal space between items
space-around --> equal space around items
space-evenly --> equal space between and around
Used with display:flex;.

4.Align items:centre

align-items controls alignment along the cross axis.

Common values:
flex-start – items align to the start
flex-end – items align to the end
center – items are centered
stretch – items stretch to fill (default)
baseline – items align by text baseline

6.Align Content:Space Around (with wrap)

align-content: space-around works in a flex container with flex-wrap: wrap. It applies vertical spacing between rows, not individual items. Equal space is added above, between, and below the rows. The container must have a fixed height and enough items to wrap onto multiple lines.

7.Order Property

The order property is used in flexbox to control the order of flex items. By default, all items have order: 0. Items with lower order values appear first. It only affects visual layout, not the HTML source order.

8.Flex Grow

The flex-grow property defines how much a flex item will grow relative to others when there is extra space. A value of 1 means the item can grow and share available space equally with other items that also have flex-grow set.

The align-self property overrides the container’s align-items for a specific flex item. It controls the item’s alignment on the cross axis (vertical by default). Values include auto, flex-start, center, flex-end, and stretch & height:auto.
